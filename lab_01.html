1. CSS Syntax ประด้วยอะไรบ้าง ?

รูปแบบของคำสั่ง CSS นั้นจะประกอบไปด้วย 3 ส่วนคือ
 1.Selector คือ ส่วน HTML tag ที่เราต้องการจะให้แสดงผล เช่น < table> ก็จะแสดงผลในส่วนของตาราง
 2.Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร , ชนิดตัวอักษร , ขนาดตัวอักษร
 3.Value คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีแดง , ตัวอักษรแบบ tahoma , ตัวอักษรขนาด 12 pt

2. การเขียน CSS มีกี่แบบ อะไรบ้าง ?
การเขียน CSS มีด้วยกัน 3 แบบ คือ

 1.inline style
 2.internal style
 3.external style

3. selector มีกี่แบบ อะไรบ้าง ?
selector แบ่งกลุ่มออกได้ 4 กลุ่ม คือ

 1.กลุ่ม ID
   rule ที่ถูกจัดให้อยู่ในกลุ่มนี้จะต้องมี id selector เป็น key selector
  
 2.กลุ่ม Class
   rule ที่ถูกจัดให้อยู่ในกลุ่มนี้จะต้องมี class selector เป็น key selector
  
 3.กลุ่ม Tag
   ถ้า key selector ของ rule นั้นไม่ใช่ทั้ง id selector และ class selector ให้ดูว่ามีการระบุ tag เอาไว้หรือไม่ ถ้ามี จะถูกจัดให้อยู่ในกลุ่ม tag
  
 3.กลุ่ม Universal
   หาก rule นั้นไม่ตรงกับ 3 กลุ่มข้างต้นเลย จะถูกจัดให้อยู่ในกลุ่ม universal
  

4. combinator มีกี่แบบ อะไรบ้าง ?
combinator แบ่งออกได้ 4 แบบ คือ
